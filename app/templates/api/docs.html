API REST — Inmobiliaria TEM742

Endpoints disponibles para consultar datos del sistema en formato JSON. Requieren autenticación con rol Administrador o Agente.

GET /api/propiedades

Lista todas las propiedades del sistema. Soporta filtros opcionales.

Parámetros opcionales:

  • tipo_id — Filtrar por tipo de propiedad (ID)
  • estado_id — Filtrar por estado (ID)
  • accion — Filtrar por acción: Venta o Alquiler
Probar Endpoint
GET /api/propiedades/<id>

Detalle completo de una propiedad específica, incluyendo datos del propietario y agente.

Ejemplo: Propiedad #1
GET /api/estadisticas

Resumen estadístico del sistema: totales de propiedades, contratos, visitas, propietarios, monto acumulado y precio promedio.

Probar Endpoint
Autenticación: Todos los endpoints requieren sesión activa con rol Administrador o Agente. Las respuestas están en formato application/json.
